# Electricity Calculator (PHP Vanilla)

This project calculates electricity power, energy, and total charge based on user input using plain PHP.

## Features
- Calculate power (Wh)
- Calculate energy (kWh)
- Calculate total electricity charge
- Per hour and per day calculation

## Technologies
- PHP (Vanilla)
- Bootstrap 4

## Formula
Power = Voltage × Current  
Energy = (Power × Hour) / 1000  
Total = Energy × (Rate / 100)

## How to Run
1. Install XAMPP and start Apache server.
2. Copy this project folder into XAMPP htdocs.
3. Open browser and go to:
   http://localhost/electricity_calculator/index.php
4. Enter values for Voltage, Current, Hours, and Rate, then click Calculate.
